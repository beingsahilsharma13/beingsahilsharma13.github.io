About

Engineering systems that
scale quietly under load.

I'm a Java backend engineer who enjoys the part of software users never see — the systems that quietly do the heavy lifting. My home turf is Java and Spring Boot: designing REST APIs, breaking monoliths into microservices, and wiring services together with Kafka so they talk asynchronously instead of blocking each other.

Right now I'm building a fintech transaction system — a microservices project with PostgreSQL for reliable storage and Redis for speed — while sharpening my data structures, algorithms and system design fundamentals every day.

I believe in understanding the why behind every tool I use. Anyone can add a dependency; knowing what it costs you is the real skill.
